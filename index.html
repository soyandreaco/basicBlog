HTML5

Hyper Text Mapkup Languaje o lenguaje de marcado de hipertexto, se refiere al código que se utiliza para crear la estructura de una página web. No es un lenguaje de programación como tal, pero si es esencial para todo lo relacionado al front de las aplicaciones web.

  1. 1995 - HTML 2.0
  2. 1997 - HTML 3.2
  3. 1999 - HTML 4.01
  4. 2007 - HTML 5.0
  5. 2014 - HTML5

CSS3

CSS por sus siglas en ingles Cascading Style Sheet, es el lenguaje de hojas de estilo que modifica la presentación de los archivos HTML.

Versiones anteriores

  2. 1998 - CSS 2
